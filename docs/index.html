Install in 30 seconds

No account, no setup wizard maze. Download, run, done. Watch the full install below.

  1. Download the installer

    Click the download button to get Folia-Setup.exe (~99 MB).

  2. Run it

    Double-click the file. Choose where to install (or accept the default) and click Install.

  3. Open Folia

    Launch it from the Start Menu or desktop shortcut and open your first PDF.

“Windows protected your PC”? Because Folia is a new, independent app it isn't code-signed yet, so SmartScreen shows a blue warning. It's safe — just click More info → Run anyway. (Prefer no install? A portable .zip is on the releases page too.)
